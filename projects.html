My Projects

Here are some of the projects I've worked on. Each one demonstrates my skills and dedication to creating high-quality applications.

Project 1: Countdown Timer App

The Countdown Timer App is a simple and user-friendly mobile application that allows users to set countdown timers for various activities, such as workouts, study sessions, and cooking. It features an intuitive interface with options to start, pause, resume, and reset the timer. When the timer reaches zero, the app provides a notification alert, ensuring you never miss a deadline.

Project 2: Travel Airline Booking Website

Project 3: Weather website

The Weather App provides real-time weather updates and forecasts for locations worldwide.
